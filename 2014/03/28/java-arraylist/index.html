Java系列之ArrayList解析

发表于 2014-03-28   |  

特性

ArrayList即实现List接口的动态数组(Resizable-array implementation of the List interface.)

  • 实现了所有list接口的操作
  • 允许元素为null
  • 时间复杂度：size、isEmpty、get、set、iterator和listIterator时间复杂度为O(1)，add和remove时间复杂度为O(n)
  • 容量：添加元素是自动增长
  • 线程同步：未实现同步
  • 迭代器：iterator和listIterator返回fail-fast迭代器（尽最大努力抛出ConcurrentModificationException）

源码分析

ArrayList继承AbstractList并实现了List接口。内部通过数组队列实现，提供了相关的添加、删除、修改、遍历等功能。实现了RandmoAccess接口，即提供了随机访问功能，ArrayList中可通过元素的序号快速获取元素对象。实现了Cloneable接口，即覆盖了函数clone()，能被克隆。实现java.io.Serializable接口，即支持序列化，能通过序列化传输。
